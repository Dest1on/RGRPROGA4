# Remember Figure Game

Windows Forms игра «Запомни фигуру» на .NET 8. Игрок тренирует внимание и реакцию: на поле случайно загораются лампочки, и по активным лампочкам нужно успеть кликнуть до того, как они погаснут.

## Возможности

- Настраиваемое количество лампочек: от 4 до 100.
- Настраиваемые интервал вспышек, длительность вспышки и количество активных лампочек за такт.
- Выбор цветов активной лампы, неактивной лампы и фона игрового поля.
- Статистика партии: всего вспышек, успешные попадания, ошибки, пропуски и процент успешности.
- Пауза, новая игра, отмена последнего изменения настроек и встроенная справка.

## Требования

- Windows 10/11.
- .NET SDK 8.0 или новее.

## Запуск

```powershell
dotnet restore
dotnet run --project RememberFigureGame/RememberFigureGame.csproj
```

## Сборка

```powershell
dotnet build RememberFigureGame.sln
```

Для Release-сборки:

```powershell
dotnet build RememberFigureGame.sln --configuration Release
```

## Горячие клавиши

| Команда | Действие |
| --- | --- |
| `Ctrl+N` | Новая игра |
| `Space` | Пауза или продолжение |
| `Ctrl+Z` | Отменить последнее изменение настроек |
| `F1` | Открыть справку |
| `Esc` | Пауза |

## Структура проекта

```text
RememberFigureGame/
  Controls/   Отрисовка игрового поля.
  Forms/      Дополнительные окна приложения.
  Models/     Состояние, настройки, статистика и лампочки.
```
